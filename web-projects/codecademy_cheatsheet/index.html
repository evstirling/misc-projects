HTML/CSS Cheat Sheet

HTML

Hypertext Markup Language

HTML, or Hypertext Markup Language is a programming language, widely used to create the structure of web content. HTML elements are formed with opening and closing tags, placing the content within them. For more information, check MDN Web Docs.

HTML tag Usage Display type Example
<h1>-<h6> Headings in descending order from 1 to 6. Block <h1>This is a title!</h1>
<p> Paragraph text block, for long sets of text. Block <p>Lorem ipsum dolor sit amet, consectetur...</p>
<a> Creates hyperlinks, specified with href. Inline <a href="https://github.com/evstirling">ev stirling</a>
<img> Embed an image and provide alt text. Block <img src="../resources/logo.png alt="Company logo." />

CSS

Cascading Style Sheets

CSS, or Cascading Style Sheets, is a method of modifiying the display of HTML elements through style rules. This allows webpages to be much more dynamic and user friendly. For more information, check MDN Web Docs.

CSS property Usage Example
color Modify text color. color: rgb(147, 108, 179);
font-family Set font type and fallback fonts. font-family: 'Times New Roman', serif;
display Modify display type - block, inline, or inline-block. display: inline-block;
opacity Modify element opacity. opacity: 0.7;

These are not exhaustive lists. Made for demonstration purposes only. Visit MDN docs, linked in the respeciive summaries, for more information.
